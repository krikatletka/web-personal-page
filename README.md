# Personal Web Page

This is my personal website project created for learning and practicing web development.

## Technologies used
- HTML
- CSS
- JavaScript

## Project structure

index.html – main page of the website  
project.html – additional page with projects  
styles.css – main styles for the website  
script.js – JavaScript functionality  

Folders:
- img/ – images used on the site
- video/ – videos used in the project
- music/ – background music
- models/ – 3D models or assets

## Features

- Responsive layout
- Dark mode
- Video background
- Interactive elements using JavaScript
- Custom design with CSS

## Purpose

The purpose of this project is to practice:
- HTML page structure
- CSS layout (Flexbox / Grid)
- JavaScript interactions
- Working with media (images, video, audio)
